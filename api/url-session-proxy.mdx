## Overview

`URLSessionProxy` is a drop-in replacement for `Foundation.URLSession` that intercepts all network activity and forwards it to `NetworkLogger`. It belongs to the `Pulse` module and requires no changes to your existing request logic — just swap out the session.

```swift
import Pulse

let session = URLSessionProxy(configuration: .default)
```

Under the hood, `URLSessionProxy` installs a `URLSessionProxyDelegate` as the session's delegate, which intercepts callbacks and records them. When `isMockingEnabled` is `true` (the default), it also prepends `MockingURLProtocol` to the configuration's `protocolClasses`, enabling request mocking in tests.

## URLSessionProtocol

`URLSessionProtocol` is a public protocol defined in the `Pulse` module. Both `URLSession` and `URLSessionProxy` conform to it:

```swift
public protocol URLSessionProtocol { ... }

extension URLSession: URLSessionProtocol {}
```

This allows you to declare your networking layer's dependency on the protocol type rather than the concrete `URLSession` class, making it straightforward to swap in `URLSessionProxy` for debug builds without changing call sites.

## Usage Example

Declare your session as `any URLSessionProtocol` (or a generic `<S: URLSessionProtocol>`) so that you can substitute `URLSessionProxy` in debug builds and `URLSession` in production:

```swift
import Foundation
import Pulse

#if DEBUG
let session: any URLSessionProtocol = URLSessionProxy(configuration: .default)
#else
let session: any URLSessionProtocol = URLSession(configuration: .default)
#endif

// Usage is identical in both cases
let (data, response) = try await session.data(for: URLRequest(url: url))
```

For an injectable networking client:

```swift
final class APIClient {
    private let session: any URLSessionProtocol

    init(session: any URLSessionProtocol = URLSession.shared) {
        self.session = session
    }

    func fetch(_ url: URL) async throws -> Data {
        let (data, _) = try await session.data(from: url)
        return data
    }
}

// In your app entry point (debug only):
#if DEBUG
let client = APIClient(session: URLSessionProxy(configuration: .default))
#else
let client = APIClient()
#endif
```

<Note>
  Completion-handler variants of `dataTask`, `uploadTask`, and `downloadTask` are handled specially: `URLSessionProxy` wraps them to call `NetworkLogger` with the response data directly, since delegate callbacks are not invoked for completion-handler tasks.
</Note>
