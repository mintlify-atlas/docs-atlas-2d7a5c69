Pulse is a powerful logging and network debugging framework for Apple platforms. It captures `URLSession` traffic, stores logs persistently on-device, and provides a native SwiftUI console you can embed directly in your app — no proxy, no external tooling required.

## How it works

<Steps>
  <Step title="Add the Pulse package">
    Add `Pulse` and `PulseUI` libraries to your app via Swift Package Manager:
    ```
    https://github.com/kean/Pulse
    ```
  </Step>
  <Step title="Capture network traffic">
    Replace `URLSession` with `URLSessionProxy` to automatically capture all network requests:
    ```swift
    import Pulse

    #if DEBUG
    let session: URLSessionProtocol = URLSessionProxy(configuration: .default)
    #else
    let session: URLSessionProtocol = URLSession(configuration: .default)
    #endif
    ```
  </Step>
  <Step title="Embed the console">
    Add `ConsoleView` to your app so your team can inspect logs on-device:
    ```swift
    import PulseUI

    NavigationLink(destination: ConsoleView()) {
        Text("Console")
    }
    ```
  </Step>
  <Step title="Connect Pulse Pro (optional)">
    Enable real-time remote logging to stream logs directly to the Pulse Pro app:
    ```swift
    #if DEBUG
    RemoteLogger.shared.isAutomaticConnectionEnabled = true
    #endif
    ```
  </Step>
</Steps>

## Supported platforms

| Platform | Minimum version |
|----------|----------------|
| iOS      | 15.0           |
| tvOS     | 15.0           |
| macOS    | 13.0           |
| watchOS  | 9.0            |
| visionOS | 1.0            |

<Note>
  Pulse is **not** a network proxy. It operates on the `URLSession` level within your app. If you need a full proxy solution, check out [Proxyman](https://proxyman.io).
</Note>
